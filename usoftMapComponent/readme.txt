1. Компонент автоматически создаёт таблицу для сохранения результатов.
2. В параметрах указывается изначальный центр карты и масштаб, ключ API-карт.
3. Нужна фронт-енд часть для отображения точек, их перемещения и передачи данных для последующего сохранения в том же формате,
в каком они переданы для отображения. Для вновь добавленных и изменённых точек должен быть проставлен флаг changed = true.
4. Компонент сделан без кеширования, т.е. время кеширования не учитывается.
